# Datart GenLab — Galeria

A galeria pública do **Datart GenLab**: arte generativa de dados, onde bases públicas
reais viram peças navegáveis e cada elemento visual volta ao dado que o gerou.

Cada obra é uma pasta. A página de entrada (`index.html`) lista todas.

## Obras
- **011 · El Niño** (`011-el-nino/`) — um globo onde a temperatura real do mar
  (NOAA OISST, 2020–2026) governa a obra; anomalia × temperatura real, lado a lado.

Publicado via GitHub Pages. Sites estáticos, rodam no navegador (Three.js + globe.gl).
